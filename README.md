# Fredriks awesome dotfiles

To get started

```shell
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply frdrkolsson
```

## Dependencies and tools

* `kitty` - terminal emulator
* `fish` - shell
* `nvim` - editor
* `tmux` - terminal multiplexer
* `bat` - cat but better
* `fd` - find but faster
* `ripgrep` - grep but faster
